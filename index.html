Student Profile WebApp

Software Engineering Class - Group 9

Project Overview

This Student Profile WebApp is a collaborative project developed by Group 9 as part of our Software Engineering coursework. The project demonstrates our ability to work together as a team to create a functional web application.

Our application allows students to create and manage their academic profiles, showcasing their skills, projects, and achievements. Each team member has contributed to different aspects of the development proces.
